Objective and Purpose
The main objective of this assignment is to apply Object Oriented Programming principles to design and implement a Java application that manages different geometric shapes. By the end of this project, the goal is to demonstrate a clear understanding of OOP concepts such as classes, inheritance, polymorphism, and encapsulation through the creation of a program that performs various geometric calculations.
The purpose of the application is to allow users to create, modify, and manage a list of shapes including rectangles, squares, triangles, and circles. The program provides basic functions such as calculating area and perimeter, translating and scaling shapes, and displaying shape information. It acts as a small geometry management system that can be extended for educational or practical use in similar graphical applications.
Methodology
The development approach focuses on a modular and incremental design, starting with the creation of a base Shape class and gradually building specialized subclasses such as Rectangle, Square, Circle, and Triangle that inherit and override their parent’s functionalities. The system includes a ShapeList class to manage collections of shapes and a main class that offers an interactive console menu.
The project uses Java as the programming language because it supports object oriented design, offers strong class structures, and is widely used for learning and applying OOP concepts. Development tools such as an IDE like IntelliJ IDEA or Eclipse and Git or GitHub for version control are used to organize code and track progress effectively.
